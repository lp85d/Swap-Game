# Swap Game 🔄

**Swap Game** — это многопользовательская логическая игра в реальном времени, в которой игроки соревнуются в скорости восстановления последовательности элементов.

## 🔗 Играть сейчас
Проект запущен и доступен по адресу: [**o1t.ru/game**](https://o1t.ru/game/)

## 🎮 Суть игры
В начале каждого раунда сервер генерирует секретную целевую комбинацию. Все игроки получают один и тот же набор элементов, но перемешанный случайным образом.

* **Цель:** Первым собрать свою комбинацию так, чтобы она полностью совпала с секретным эталоном сервера.
* **Механика:** Игроки ходят по очереди, меняя местами (swap) любые два элемента в своем ряду.
* **Особенность:** Игроки не знают правильную последовательность заранее. Победа зависит от логики, памяти и скорости принятия решений.

## 🚀 Технический стек
* **Backend:** PHP 8.2 + Ratchet (WebSockets).
* **Runtime:** ReactPHP Event Loop.
* **Web Server:** Nginx (Reverse Proxy + SSL/WSS).
* **Frontend:** Pure JavaScript (Vanilla JS).

## 🛠 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/lp85d/Swap-Game.git](https://github.com/lp85d/Swap-Game.git)
    ```
2.  **Установите зависимости:**
    ```bash
    composer install
    ```
3.  **Запустите сервер:**
    ```bash
    php server.php
    ```

## 📈 Логирование и администрирование
Сервер выводит подробную информацию о текущей сессии в консоль:
* Сгенерированный правильный ответ (Target Goal).
* Начальные наборы каждого игрока.
* Лог каждого совершенного хода в реальном времени.
