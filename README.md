# Swap Game 🔄

**Swap Game** — это многопользовательская логическая головоломка в реальном времени, построенная на механике перестановок. Задача игроков — привести свою хаотичную последовательность элементов к секретному эталону, который знает только сервер.

## 🎮 Суть игры
В начале раунда сервер генерирует скрытую «целевую комбинацию» (Goal). Каждому игроку выдается тот же набор элементов, но в случайном порядке.
* **Цель:** Первым восстановить целевую последовательность.
* **Механика:** Игроки ходят по очереди, меняя местами любые два элемента в своем наборе.
* **Сложность:** Игроки не видят правильный ответ. Побеждает тот, кто быстрее вычислит верный порядок или проявит лучшую интуицию.

## 🚀 Технологии
Проект реализован на современном стеке для обеспечения высокой скорости работы через WebSockets:
* **Backend:** PHP 8.2 + [Ratchet](http://socketo.me/) (библиотека для WebSockets).
* **Server Runtime:** ReactPHP (Event Loop).
* **Web Server:** Nginx (в качестве Reverse Proxy с поддержкой SSL/WSS).
* **Frontend:** Чистый JavaScript (Vanilla JS), CSS3, HTML5.

## 🛠 Архитектура
1.  **Nginx:** Принимает зашифрованный трафик (WSS на порту 443) и проксирует его на локальный порт WebSocket-сервера.
2.  **WebSocket Server (`server.php`):** Полноценный игровой движок, который хранит состояние сессии, проверяет очередность ходов и сверяет комбинации в памяти.
3.  **Client (`index.php`):** Легкий интерфейс, который взаимодействует с сервером по протоколу JSON.
